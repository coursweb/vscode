## Désactiver l'IA

Il est possible de désactiver les auto-suggestions utilisant l'IA.

Dans l'interface:

Sous **Conversation > Divers** : *Disable AI Features*

Dans le JSON, cela correspond au code suivant:

```
"chat.disableAIFeatures": true,
```
